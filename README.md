# 🕷️ Hacker News Web Scraper

A Python command-line tool that scrapes top stories from Hacker News and lets you search, filter and save them to CSV.

## ✨ Features

- Scrapes top 30 stories from Hacker News
- Shows rank, title, score and comments
- Search stories by keyword
- Save all stories to CSV file
- No external libraries needed!

## 🛠️ Tech Stack

- Python 3
- urllib (built-in)
- csv (built-in)
- re — Regular Expressions (built-in)

## ▶️ How to Run
```bash
python web_scraper.py
```

## 📋 Menu Options

| Option | Action |
|--------|--------|
| 1 | Show top 10 stories |
| 2 | Show top 20 stories |
| 3 | Search by keyword |
| 4 | Save to CSV |
| 5 | Exit |
